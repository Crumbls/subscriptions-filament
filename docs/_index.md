`crumbls/subscriptions-filament` adds Filament v5 admin resources for the `crumbls/subscriptions` package. Plans, Features, and Subscriptions get full CRUD with relation managers, status badges, and inline cancel / renew actions. A drop-in `PlanSubscriptionsRelationManager` lets you bolt the subscription tab onto any subscriber resource (User, Tenant, Team).
